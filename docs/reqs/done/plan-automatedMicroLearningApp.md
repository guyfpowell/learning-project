# Automated Micro-Learning App — Full Business & Tech Plan

## Executive Summary

Build an **AI-native micro-learning platform** that generates, personalizes, and adapts daily 3-minute lessons in real-time. This is not a static content library—it's an AI co-pilot that dynamically creates and tailors lessons, quizzes, and coaching prompts based on each user's learning goals, pace, and application context. 

**Your unique angle**: Leverage your 20 years of PM/product leadership expertise to:
- Build the product for product leaders, PMs, and AI practitioners first (eating your own dog food)
- Position as the only AI-powered microlearning platform, not a content app
- Create enterprise/team versions with AI-driven team learning analytics
- Use your credibility to establish thought leadership and B2B partnerships

**Differentiation**: AI-generated lessons (not curated), real-time adaptation (not static paths), and PM/AI product-focused vertical (not generic skills).

## 3. Product Concept

### Core Idea
- An **AI-native micro-learning platform** that generates, personalizes, and adapts lessons in real-time
- Not a content library—a true AI learning co-pilot
- Focus on learning outcomes + skill application + AI-driven personalization

### How It Works
1. User sets initial goal (e.g., "Learn AI product strategy")
2. AI generates personalized daily lesson based on goal + previous progress
3. User completes 3-minute lesson + quick quiz
4. AI evaluates understanding and generates next lesson (may adapt difficulty, topic, or style)
5. Real-time feedback and next steps based on AI analysis
6. System learns user's pace, learning style, and knowledge gaps

### Key Differentiators
1. **AI-generated lessons (not curated curated)**: LLM-powered content generation and variation
   - Each lesson is generated fresh based on user context, learning goals, and pacing
   - Real-time adaptation to quiz performance and engagement patterns
2. **AI personalization engine** with multi-armed bandits
   - Next lesson selected by AI, not static paths
   - Real-time learning style detection and adaptation
3. **AI coaching and feedback loops**
   - Personalized guidance and micro-coaching based on quiz answers
   - Real-time hints and explanations tailored to knowledge gaps
4. **Product leadership + AI vertical focus**
   - Lessons on AI product strategy, governance, LLM integration, prompt engineering, AI ethics
   - First target audience: PMs, product leaders, and AI practitioners
5. **Enterprise team learning analytics**
   - AI-powered insights into team skill gaps and learning velocity
   - Cohort recommendations and team challenges

### Primary Target Audience (Phase 1)
- **Product managers** (early-career to director level) learning AI product best practices
- **Product leaders** acquiring skills in prompt engineering, AI governance, and AI-first product strategy
- **AI practitioners** wanting bite-sized learning on latest AI product patterns and trends
- **Startup founders** building AI products and needing rapid skill development

### Secondary Audiences (Phase 2+)
- Enterprise product teams licensing the platform for team learning
- Larger organizations using AI-powered team skill assessments and recommendations
- General professionals (existing market) once platform is proven

## 7. Full Tech Implementation Plan (Cost-Optimized for Zero/Low AI Bills)

### AI Cost Strategy — Three Tiers

#### Tier 1: Free/Bootstrap (MVP)
- **Model**: Ollama + Mistral 7B or Llama 2 13B (self-hosted)
- **Inference cost**: $0 (runs on $10–30/month GPU)
- **Approach**:
  - Pre-generate 5 lesson templates per skill weekly
  - Remix templates with variable substitution (topic, examples, user context) = 50+ variations per week, zero LLM cost
  - Use open-source embeddings (sentence-transformers) for recommendations
  - Multi-armed bandit for personalization (no AI needed)
- **Quality**: 85% of GPT-4, perfectly fine for PM/AI learner audience
- **Best for**: MVP, early validation, bootstrapping

#### Tier 2: Hybrid (Scale)
- **Model**: Mistral 7B (free) for fast generation + OpenAI API key provided by user
- **Approach**:
  - Core lessons: generated via self-hosted Mistral (free)
  - Premium features (advanced coaching, quiz explanations): use user's own OpenAI key
  - "Bring your own API key" model
- **Cost to you**: $0
- **Cost to users**: Only if they want premium personalization
- **Unit economics**: Better—users pay for GPT-4 if they want it
- **Best for**: Scaling B2C while keeping your costs zero

#### Tier 3: Premium AI (Optional, User-Paid)
- **Model**: OpenAI, Anthropic, or Mistral API
- **Approach**: User pays for "AI concierge" mode (human-quality explanations, advanced coaching)
- **Revenue play**: Premium tier that includes API costs
- **Example**: $49/month user tier includes $5/month in GPT-4 calls; you absorb cost, user gets premium experience

### Architecture Overview (Cost-Optimized)

#### Frontend
- **Mobile**: React Native or Flutter
  - Reason: cross-platform, fast iteration, strong habit/notification features
- **Web**: React + Next.js
  - Dashboard, onboarding, account management

#### Backend
- **API Server**: Node.js + NestJS (or Express)
  - User auth, profiles, subscription, lesson delivery
- **Database**: PostgreSQL
  - Users, lessons, progress, subscriptions
- **Cache/Session**: Redis
  - Session management, real-time streak tracking

#### Content Management
- **Headless CMS**: Sanity, Strapi, or custom admin UI
  - Author lessons, manage skill paths, versioning

#### AI & Personalization (Zero-Cost Design)
- **Lesson generation**: Ollama + Mistral 7B (runs on $10/month GPU instance)
  - Template-based generation: create 5 templates, remix 50+ variations via substitution
  - Cost: 0 after $10 GPU rental
- **Recommendation engine**: Multi-armed bandit (no LLM needed)
  - Track which lessons have highest engagement/quiz scores for each user
  - Select next lesson based on historical performance, not AI
  - Pure math, not API calls
- **Quiz generation**: Template-based (no AI)
  - Create 10 quiz templates per topic
  - Vary answers and distractors via template substitution
  - Zero cost
- **Adaptive difficulty**: Bayesian skill tracking (no LLM)
  - Track user's estimated knowledge level from quiz performance
  - Adjust next lesson difficulty based on Elo-style rating
  - No AI calls required
- **Fallback to paid**: If user enables premium AI coaching, use their OpenAI API key
  - You provide UI/UX; they pay for tokens
  - Or use free Replicate API with rate limits

#### Payments
- **Stripe**: subscriptions, one-time bundles, trial management

#### Notifications
- **Push notifications**: Firebase Cloud Messaging or OneSignal
- **Email**: SendGrid or Postmark

#### Hosting & Infrastructure
- **Backend**: Vercel, AWS Elastic Beanstalk, or DigitalOcean App Platform
- **Database**: Supabase, AWS RDS, or managed PostgreSQL
- **Storage**: AWS S3 or DigitalOcean Spaces
- **CDN**: Cloudflare or AWS CloudFront (for lesson assets)

**Monorepo benefits**:
- **Shared types**: `packages/shared` consumed by `api`, `web`, and mobile
- **Single source of truth**: API contract defined once, used everywhere
- **Easy refactoring**: Change an API type, get errors in all consumers
- **Unified CI/CD**: One GitHub repo, one set of workflows
- **Local development**: `pnpm install` installs all packages at once
- **Prisma schema**: Single database schema, shared by all services

**Development workflow**:
```bash
# Install dependencies (all packages)
pnpm install

# Start local dev (Docker Compose starts API, PostgreSQL, Redis)
docker-compose up

# OR run individual packages for debugging
pnpm --filter=api dev
pnpm --filter=web dev

# Deploying
git push → GitHub Actions:
  - Tests run
  - If tests pass, API deploys to Render
  - Web deploys to Vercel
```

## Appendix: Tech Stack Recommendations

### Recommended Stack (AI-Native First)
Better than traditional—optimize for rapid AI iteration and scaling.

### Frontend
- **Web**: React (TypeScript) + Next.js + Tailwind CSS + Shadcn UI
  - Enterprise-grade type safety
  - Server-side rendering for performance
  - Type-safe API integration
- **Mobile**: React Native (TypeScript) with Expo
  - Shared component logic with web (TypeScript types across platforms)
  - Expo Go for rapid iteration in development
  - Native performance without managing Xcode/Android Studio
  - Same TypeScript codebase patterns

### Backend
- **Runtime**: Node.js (TypeScript)
- **Framework**: Express (TypeScript) with TypeScript middleware
  - Enterprise-grade type safety across API layer
  - Lightweight, proven SaaS patterns
  - Easy to scale horizontally
- **ORM**: Prisma (TypeScript)
  - Type-safe database queries
  - Auto-migrations and schema management
  - Works perfectly with PostgreSQL
- **Database**: PostgreSQL (relational, powerful; store lessons, quiz answers, user progress)
  - Run in Docker container
  - Managed via Prisma migrations
- **Cache**: Redis (streaks, sessions, real-time user state)
  - Also in Docker container
- **Job queue**: Bull (Node.js queuing library) for async lesson generation jobs
  - Integrates cleanly with Node.js/Express
- **Containerization**: Docker for all services
  - Separate containers for: Node.js API, PostgreSQL, Redis
  - Docker Compose for local development (all services up with `docker-compose up`)

### Infrastructure (Chosen Stack)
- **Containerization**: Docker + Docker Compose
  - All services containerized for reproducibility
  - Local dev environment = production environment
  - Easy CI/CD pipeline integration (GitHub Actions with Docker)
- **Frontend Hosting**: 
  - **Next.js**: Vercel (native Next.js support, zero-config deployments)
    - Automatic deployments on Git push
    - Edge functions and serverless functions included
    - Free SSL/TLS, CDN, and analytics
  - **React Native**: Deployed as native apps to App Store/Google Play
- **Backend Hosting**: 
  - **Render**: Node.js/Express API
    - Native Docker support (push Dockerfile, auto-deploys)
    - Environment variables, auto-scaling, monitoring
    - ~$7/month hobby tier or $12+/month for production
- **Database**: 
  - **Supabase** (managed PostgreSQL + Auth + Realtime)
    - Hosted PostgreSQL with automatic backups
    - Schema managed via Prisma migrations
    - Free tier: 500MB storage, 2 concurrent connections
    - Pro tier: ~$25/month (unlimited connections, more storage)
    - Prisma integrates seamlessly with Supabase
- **Cache**: 
  - **Render Redis**
    - Render-hosted Redis (same platform as API for simplicity)
    - ~$7/month for shared tier or higher for dedicated
    - Or use Upstash (serverless Redis) ~$5/month if cost is priority
- **AI/ML Layer** (Cost-Optimized):
  - **Ollama + Mistral 7B**: Run in separate Docker container on $10–30/month GPU instance
  - Accessible to Node.js API via HTTP calls
  - Shared infrastructure pattern: one Docker container for Ollama, queried from main API
- **Storage**: S3 or Backblaze B2 (lesson assets, user exports)
  - Accessed from Node.js API via AWS SDK
- **CDN**: Cloudflare (free tier)
- **Monitoring**: Sentry (error tracking), DataDog or New Relic (optional; free tiers available)
